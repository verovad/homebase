# 🏠 HomeBase

Aplicación web de gestión del hogar familiar desarrollada con Python y Flask.

## ¿Qué hace?

- 💰 **Gastos**: Registra y categoriza los gastos del hogar
- 💵 **Ingresos**: Controla los ingresos familiares
- 📅 **Agenda**: Calendario compartido con eventos y recordatorios
- 🎯 **Objetivos**: Metas de ahorro con barra de progreso

## Tecnologías utilizadas

- Python 3.14
- Flask
- SQLite
- HTML5 / CSS3

## Cómo ejecutarlo

1. Clona el repositorio
2. Crea un entorno virtual: `python -m venv venv`
3. Actívalo: `venv\Scripts\activate`
4. Instala dependencias: `pip install flask`
5. Inicializa la base de datos: `python database.py`
6. Arranca la app: `python app.py`
7. Abre `http://127.0.0.1:5000`
